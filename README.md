# kareem-portfolio

Portfolio site for **Kareem Meenazi** — Business Analyst & Data Enthusiast.

Built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Live Site

[kareem-portfolio-psi.vercel.app](https://kareem-portfolio-psi.vercel.app)

## Sections

- **Hero** — Introduction with animated profile card
- **About** — Background, focus areas, education, and status
- **Skills** — Core soft skills, technical/business skills, and tools
- **Experience** — MUN, FMCG case studies, Sales, Real Estate
- **Pursuits** — Personal endeavors and extracurricular activities
- **Contact** — LinkedIn, WhatsApp, Email

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Deployed on Vercel
